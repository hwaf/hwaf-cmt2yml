hwaf-cmt2yml
============

This is a simple WIP converter of `CMT` requirements files into
(eventually) `YAML` based files.

## Example

```sh
$ hwaf cmt2yml
::: hwaf-cmt2yml
>>> dir="."
::> [CTPfragment/cmt/requirements]...
::> [EventApps/cmt/requirements]...
::> [EventStorage/cmt/requirements]...
::> [MuCalDecode/cmt/requirements]...
::> [TDAQCExternal/Boost/cmt/requirements]...
::> [TDAQCExternal/Python/cmt/requirements]...
::> [TDAQCExternal/cmt/requirements]...
::> [TDAQCExternal/dCache/cmt/requirements]...
::> [TDAQCExternal/uuid/cmt/requirements]...
::> [TDAQCExternal/zlib/cmt/requirements]...
::> [TDAQCPolicy/TDAQCPolicyExt/cmt/requirements]...
::> [TDAQCPolicy/TDAQCPolicyInt/cmt/requirements]...
::> [TDAQCPolicy/cmt/requirements]...
::> [TDAQCRelease/cmt/requirements]...
::> [circ/cmt/requirements]...
::> [clocks/cmt/requirements]...
::> [eformat/cmt/requirements]...
::> [ers/cmt/requirements]...
::> [toto/requirements]...
req="CTPfragment/cmt/requirements"
req="CTPfragment/cmt/requirements" [done]
req="EventApps/cmt/requirements"
req="EventApps/cmt/requirements" [done]
req="EventStorage/cmt/requirements"
req="EventStorage/cmt/requirements" [done]
req="MuCalDecode/cmt/requirements"
req="MuCalDecode/cmt/requirements" [done]
[...]
```
